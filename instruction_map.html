📘 Gantt Chart Symbols Guide

  • ◆ – Start of phase or initiative
  • ● – Key milestone (e.g., MVP delivery, campaign launch)
  • → – Task with a dependency on a prior task
  • Bold Phase Title – Indicates a grouped project phase
  • Smaller text – Provides details or clarifications on the task

Hover over any task bar in the Gantt charts to see more information.
